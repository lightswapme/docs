LightSwap Partners is the public documentation hub for teams that want to add LightSwap to their product.

Today, the live integration path is backend API usage. Widget integration is planned and will be added here when it is ready.

<Note>
  The API is server-to-server. Keep API keys on your backend and out of browser code, mobile apps, public repositories, and client-side logs.
</Note>

If you need API access, onboarding, fee sharing, or want to discuss widget plans, email [partner@laptev-fzco.com](mailto:partner@laptev-fzco.com).
