LightSwap documentation helps teams add swaps to their product with LightSwap.

You can integrate through the backend API or place the hosted widget on your website with an iframe.

<Note>
  The API is server-to-server. Keep API keys on your backend and out of browser code, mobile apps, public repositories, and client-side logs.
</Note>

If you need API access, onboarding, fee sharing, or a public widget key, [contact LightSwap](https://lightswap.me/support).
