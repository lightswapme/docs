Use this page when you need help with a live LightSwap integration or with a specific swap created through your integration.

## Use partner email instead

Do not use this page as the primary route for:

- API access
- onboarding
- fee sharing
- widget interest

Use the LightSwap partner team email for those cases:

<Card title="Email partner team" href="mailto:partner@laptev-fzco.com">
  partner@laptev-fzco.com
</Card>

## When to contact support

Use support for:

- existing integration questions
- unclear API behavior
- swap status questions
- payout or deposit follow-up on a specific swap

## What to include

Include as much of the following as you have:

- your `externalId`
- LightSwap swap ID, if one exists
- deposit transaction hash, if funds were already sent

This helps LightSwap find the request or swap faster.

## Integration help

If your question is about an existing integration, include:

- the integration type you are using today
- the endpoint or page you are working on
- the exact error code, if available

## Transaction help

If your question is about a specific swap, include:

- the swap ID
- the deposit transaction hash
- the latest status you received

## What not to send

Do not send:

- private keys
- seed phrases
- raw secret values
- full internal customer records
