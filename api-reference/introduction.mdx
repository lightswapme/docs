The API reference is generated from the LightSwap Partner API OpenAPI specification.

## Base URL

```text
https://lightswap.me/api/partner/v1
```

## Authentication

Every request requires the `x-api-key` header:

```http
x-api-key: <partner_api_key>
```

Every response includes an `x-request-id` header. Include that value when you contact support about a request.

## Response format

Successful responses return the endpoint payload directly. Error responses use a shared structure:

```json
{
  "error": {
    "code": "quote_expired",
    "message": "Partner quote has expired.",
    "details": {}
  },
  "requestId": "request-id"
}
```
